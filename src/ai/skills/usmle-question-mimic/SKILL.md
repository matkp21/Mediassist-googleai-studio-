# USMLE Question Mimic

Generates high-quality, board-style (USMLE/PLAB/NEET-PG) clinical vignettes using Progressive Disclosure. It dynamically reads from `references/nbme-style-guide.md` to format the vignette strictly according to international medical board examinations, completely avoiding buzzwords and forcing deep clinical reasoning.
