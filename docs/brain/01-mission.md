# Mission
MediAssistant is an AI-Powered Medical Assistant designed to provide a comprehensive study ecosystem for medical students (Medico Mode) and advanced tools for healthcare professionals (Pro Mode).

Key Goals:
1. Seamless, crash-free user experience.
2. Accurate, AI-driven medical insights using Gemini/MedGemma.
3. Strict adherence to Next.js App Router and React best practices.
