# Lab Value Pathology Mapper

## Purpose
Instead of simply stating "Sodium is low," this skill forces the AI into a pedagogical Socratic workflow. It asks the student to explain the *underlying pathophysiology* causing the abnormal lab values before providing the answer.

## Progressive Disclosure workflow
Loads tables of common laboratory reference ranges and pathophysiological associations exclusively when evaluating biochemical arrays.
