# PubMed EBM Synthesizer

## Purpose
Instead of retrieving and dumping dense research abstracts, this skill automatically synthesizes the retrieved Evidence-Based Medicine (EBM) data into a "Bottom Line Up Front" (BLUF) clinical summary, perfectly tuned for rapid ward studying.

## Progressive Disclosure workflow
Enhances the external Model Context Protocol (MCP) server for literature databases. Only loaded when processing specific external PubMed retrieval operations.
